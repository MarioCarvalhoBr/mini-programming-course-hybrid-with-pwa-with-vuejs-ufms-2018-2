CSS - Cascading Style Sheets

Cascading Style Sheets é um termo técnico que foi traduzido para a língua portuguesa como "folhas de estilo em cascata", é uma técnica de linguagem de programação web para adicionar estilo a um documento web. Em vez de colocar a formatação dentro do documento, o CSS cria um link para uma página que contém os estilos.
